color picker (under construction)

_g.color is a javascript script built to render a color picking tool and return the hexadecimal code of the chosen color. the script was originally designed for Graphene and therefore uses the conventions for related _g scripts.



scripts.phene.co



getting started

the first step in setting up the color picker is to include the library in the html file. the source file can be downloaded here and put on a cdn or use the following script tag to include it directly from phene.co: 

<script src="http://scripts.phene.co/color/0.1.0.0008"></script>

the second step in setting up the color picker is creating an element for the tool to be housed in. whatever element you pass to the tool will be treated as the base of the tool, being filled with all the necessary elements to render the interface. here's a basic example: 

<div id="color"></div>

_g.color is a class included in the Graphene or _g object. it can be accessed with either _g.color or _g.cl. in all instances _g can be replaced with Graphene for improved readability. calling the script is done in the following form: 

var element	= document.getElementById('color'),
	options	= {},
	color	= new _g.color.picker(element, options);

this is the most basic example, making use of none of the built-in options and rendering just a normal picker. do note, however, that due to this page's stylesregarding tables, some extra css has been added in the header to fix some issues, this may be required for other pages as well.
